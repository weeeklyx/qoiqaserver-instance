# Сборка QoiqaServer
Сбалансированный геймплей, основанный на технолиях и магии.

## Идея
Модпак основан на моде Create и Irons's Spells.

## Особенности
### Оптимизация

Плавный геймплей даже на слабых компьютерах.
###Тестировалось на:
GT 710 & Core I3 3200 (50-60 FPS)
GTX 1660 SUPER & Ryzen 5500 (100-200 FPS)
RTX 4060 & I5 13400F (300+ FPS)

### Больше 100 модов
Несколько примеров:

- Create и аддоны к нему
- Iron's Spells
- Farmer's Delight
- Let's Do
